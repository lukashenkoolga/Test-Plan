# Сценарии заполнения формы записи

## Позитивные сценарии

## Сценарий №1

### Шаги воспроизведения

1. Заполнить поле "Имя" кириллицей от 2 до 20 символов
2. Заполнить поле "Телефон" +7 (10 любых цифр)
3. Заполнить поле "Электорнная почта" латинскими буквами с использованием символа @
4. Кликнуть на кнопку "Записаться"

### Ожидаемый результат:

Запись прошла успешно. Получено сообщение "С Вами свяжется менеджер".

# Перечень используемых инструментов

* **IntelliJ IDEA** - программа, позволяющая написать код.
* **Java 11** - язык программирования.
* **JUnit5** - среда тестирования для приложений Java.
* **Gradle** - система сборки для упрощения работы с Java.
* **Selenide** - фреймворк для автоматизированного тестирования веб-приложений.
* **Faker** - генерация пользовательских данных при тестированиии.
* **Allure** - фреймворк для сбора данных и постоения отчетов о тестировании кода.

# Перечень необходимых разрешений/данных/доступов

* Разрешение на автоматизацию
* Доступ в тестовое приложение
* Набор тестовых данных
* Спецификации
* Доступ к базе данных

# Перечень и описание возможных рисков при автоматизации

* Ошибки в коде
* Проблемы с поиском локаторов для нахождения элементов на странице
* Большая нагрузка на сайт

# Перечень необходимых специалистов для автоматизации

* Инженер по автоматизированному тестированию

# Интервальная оценка с учетом рисков

* Подготовка и настройка необходимых инструментов - 6 часов;
* Необходимое время на тестирование с учетом рисков - 14 часов;
* Работа с багами - 5 часов;
* Подготовка отчетов - 4 часа.

Итого - 29 часов.
